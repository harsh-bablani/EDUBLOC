# EDUBLOC

A comprehensive platform leveraging blockchain for secure, transparent, and efficient educational management. This repository facilitates decentralized certificate issuance, student record management, and other educational operations using blockchain and modern web technologies.

---

## üöÄ Getting Started

### 1. Clone the Repository

```
git clone https://github.com/harsh-bablani/EDUBLOC.git
cd EDUBLOC
```

### 2. Set Up Your Environment

Create a Python virtual environment (if backend is in Python):

```
python -m venv venv
source venv/bin/activate       # On Windows: venv\Scripts\activate
```

Or set up Node.js dependencies if the project uses JavaScript/TypeScript:

```
npm install
```

### 3. Install Dependencies

If using Python:

```
pip install -r requirements.txt
```

If using Node.js:

```
npm install
```
